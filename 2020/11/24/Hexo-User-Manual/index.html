支持 GraphViz 画图

使用hexo-filter-viz插件来绘制。

安装：npm install https://github.com/Gowa2017/hexo-filter-viz.git

支持把dot或viz代码块渲染成graphviz图形。可以通过 {engine=”…”} 来选择不同的渲染引擎。 引擎 circo，dot，neato，osage，或者 twopi 是被支持的。默认下，使用 dot 引擎。

参考：https://shd101wyy.github.io/markdown-preview-enhanced/#/diagrams?id=graphviz

但是在开启pjax时，graphviz图形需要手动刷新才能显示，所以不得不关闭pjax。
