Node Shapes

еёёз”ЁеұһжҖ§д»Ӣз»Қ

иҠӮзӮ№пјҲnodeпјүеұһжҖ§

иҠӮзӮ№зҡ„еҪўзҠ¶дё»иҰҒжңүдёүз§ҚпјҡPolygon-based Nodes, Record-based Nodes, User-defined NodesгҖӮзӮ№еҮ»й“ҫжҺҘеҸҜзңӢеҲ°иҜҰз»Ҷд»Ӣз»ҚгҖӮ

еұһжҖ§еҗҚз§° д»Ӣз»Қ
shape иҠӮзӮ№еҪўзҠ¶пјҢboxй•ҝж–№еҪўгҖҒcircleеңҶеҪўзӯүпјҢеҸӮиҖғNode Shapes
color иҫ№жЎҶйўңиүІ
style иҠӮзӮ№ж ·ејҸпјҢfilled, invisible, diagonalsзӯүпјҢеҸӮиҖғStyles for Nodes
fillcolor еЎ«е……йўңиүІ, еҸӘжңүеңЁstyle=filledж—¶жүҚз”ҹж•Ҳ
fontcolor ж–Үеӯ—йўңиүІ

иҝһжҺҘзәҝпјҲedgeпјүеұһжҖ§

еұһжҖ§еҗҚз§° д»Ӣз»Қ
label ж–Үеӯ—жҸҸиҝ°
headlabel иҝһжҺҘзәҝеӨҙйғЁж–Үеӯ—жҸҸиҝ°пјҢжҜ”еҰӮA -> B, Aз«ҜжҳҜе°ҫйғЁпјҢBз«ҜжҳҜеӨҙйғЁ
taillabel иҝһжҺҘзәҝе°ҫйғЁж–Үеӯ—жҸҸиҝ°
headport иҝһжҺҘзәҝеӨҙйғЁжҢҮеҗ‘иҠӮзӮ№зҡ„дҪҚзҪ®пјҢn(north), s(south), w(west), e(east)
tailport иҝһжҺҘзәҝе°ҫйғЁжҢҮеҗ‘иҠӮзӮ№зҡ„дҪҚзҪ®
color иҝһжҺҘзәҝйўңиүІ
labelfontcolor ж–Үеӯ—йўңиүІпјҢиҝҳжңүfontnameе’Ң
arrowhead з®ӯеӨҙж ·ејҸ
splines иҝһжҺҘзәҝж ·ејҸпјҢзӣҙзәҝгҖҒжӣІзәҝгҖҒжҠҳзәҝзӯүпјҢsplines=falseи®ҫзҪ®дёәзӣҙзәҝгҖӮ
dir иҝһжҺҘзәҝж–№еҗ‘
headclip иҝһжҺҘзәҝеӨҙйғЁжҢҮеҗ‘иҠӮзӮ№иҫ№зјҳжҲ–иҠӮзӮ№дёӯеҝғпјҢй»ҳи®ӨжҳҜиҫ№зјҳ
tailclip иҝһжҺҘзәҝе°ҫйғЁжҢҮеҗ‘иҠӮзӮ№иҫ№зјҳжҲ–иҠӮзӮ№дёӯеҝғпјҢй»ҳи®ӨжҳҜиҫ№зјҳ
weight иҝһжҺҘзәҝеёғеұҖзі»ж•°пјҢеҖји¶ҠеӨ§еҲҷзәҝи¶ҠзҹӯгҖҒи¶ҠзӣҙгҖҒи¶ҠеһӮзӣҙ
constraint дёәfalseж—¶пјҢиҜҘиҝһжҺҘзәҝдёҚз”ЁдәҺиҠӮзӮ№еёғеұҖзҡ„и®Ўз®—
decorate дёәtrueж—¶пјҢиҝһжҺҘзәҝж ҮзӯҫйҖҡиҝҮдёӢеҲ’зәҝйҷ„зқҖеҲ°иҝһжҺҘзәҝдёҠ

еёғеұҖеұһжҖ§

еұһжҖ§еҗҚз§° д»Ӣз»Қ
rankdir еӣҫеҪўеёғеұҖж–№еҗ‘пјҢй»ҳи®ӨTBпјҢиЎЁзӨәtop -> bottomгҖӮеҖјдёәLRиЎЁзӨәleft -> rightгҖӮ
ranksep жңҖе°Ҹй—ҙйҡ”
group еҜ№иҠӮзӮ№жңүж•ҲпјҢеҗҢз»„иҠӮзӮ№зҡ„иҝһжҺҘзәҝе°ҶдҝқжҢҒеңЁдёҖдёӘзӣҙзәҝж–№еҗ‘гҖӮзӨәдҫӢ

и®ҫзҪ®nodeе’Ңedgeзҡ„й»ҳи®ӨеұһжҖ§

nodeе’ҢedgeжҳҜдёӨдёӘзү№ж®Ҡе…ій”®еӯ—пјҢз”ЁдәҺи®ҫзҪ®й»ҳи®ӨеұһжҖ§

еӨҡзӮ№иҝһзәҝ

жҺ§еҲ¶е…ғзҙ ж°ҙе№іеҜ№йҪҗ
